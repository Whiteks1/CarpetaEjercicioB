Uso de Fetch para acceder datos externos

Un ejemplo de conecxi√≥n a un origen de datos externos (github) que devuelve un JSON que vamos a usar algunos datos para llenar el contenido del DOM


            async function cargarDatos() {
                const url = 'https://api.github.com/users';

                try {
                    const res = await fetch(url);

                    if (!res.ok) {
                        throw new Error('No se puede conectar con el servidor');
                    }

                    const datos = await res.json();
                    console.log(datos)
                    //enviamos los datos a procesar
                    procesarDatos(datos, 'usuarios')

                } catch (error) {
                    console.error(error.message);
                }
            }
